Design review · For Tree

The stage diagram, five ways. Pick your favorite.

Five treatments of the “Build the Leader → Build the Team → Build Future Leaders” diagram, each shown exactly where it will live: the Operating System section on the TLC for Leaders page, and the Formula section (with the outcome box) on the TLC for Organizations page. Every version uses the site's existing colors and type, and every version supports what the admin needs: edit stage titles, run 3 or 4 stages, and set each page's stages independently.

Options are ordered calm → bold — use the “4 stages” toggle to preview admin flexibility.
What happens next: Tree picks an option (or a mix — e.g. “Option B, but with Option D's outcome card”). We then build it as a real component wired to the admin console, so stage names, the number of stages (3 or 4), and the outcome text are all editable per page without touching code.
